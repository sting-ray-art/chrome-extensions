# デイコード 参加者フィルター（Chrome拡張）

[デイコード](https://character-sheets.appspot.com/schedule) の **日程状況一覧** ページで、ホワイトリスト方式により選択した参加者の列だけを表示します。合計列（全・昼・夜・×・－）も、表示中の参加者だけで再計算されます。

TRPG など参加者が多いセッションで、特定メンバーだけの空き状況を確認したいときに使えます。

---

## このツールでできること

- 日程表の参加者列を、チェックしたユーザーだけ表示
- **名前検索** で参加者リストを絞り込み
- 参加者リストを **50音順** で表示（先頭が数字の名前は上に表示）
- 設定は **セッション（URL の key）ごと** に Chrome へ保存
- 表示対象外の参加者を「日程入力」プルダウンからも非表示

## 対象ページ

次の条件を満たすページで動作します。

- URL が `https://character-sheets.appspot.com/schedule/list?key=...`
- ページ内に `table.schedule_table` がある

## 使い方

### 基本操作

1. Chrome でデイコードの **日程状況一覧** を開く
2. 画面右下に **「参加者フィルター」** パネルが表示される
3. 表示したい参加者にチェックを入れる（ホワイトリスト）
4. チェックを外した参加者の列は表から隠れる
5. 設定は自動保存され、次回同じセッション（同じ `key`）を開いたときに復元される

### 名前検索

1. パネル上部の **「名前で検索...」** にキーワードを入力する
2. 参加者リストには **条件に一致した名前だけ** が表示される
3. **日程表の表示は検索に影響されない**（チェック状態だけで制御）
4. 検索中にリストから消えた参加者でも、チェック済みなら表には表示されたまま
5. 検索を空にすると、全参加者がリストに再表示される
6. 該当者がいない場合は「該当する参加者がいません」と表示される

### 表示順

参加者リストは次のルールで並びます。

1. **先頭が数字**（`0`〜`9`、全角数字を含む）の名前を上に表示
2. それ以外の名前を下に表示
3. 各グループ内は **50音順**（日本語ロケール）

※ 日程表本体の列順は変更しません。並び替えはパネル内のリストのみです。

### パネルのボタン

| ボタン | 機能 |
|--------|------|
| 表示中をすべて選択 | 検索結果に表示されている参加者をすべてチェック |
| 表示中をすべて解除 | 検索結果に表示されている参加者のチェックをすべて外す |
| − / ▢ | パネルの折りたたみ |

## うまく動かないとき

### パネルが表示されない

- **一覧ページ以外**（候補日設定・入力画面など）では表示されません
- ページを再読み込み（F5 キー、または Ctrl+R / Cmd+R）してみてください
- `chrome://extensions` を開き、「デイコード 参加者フィルター」が **有効** になっているか確認してください

### 拡張機能を入れ直したい

[ルート README の「更新版を入れるとき」](../README.md#更新版を入れるとき) の手順に従ってください。

## 技術メモ（開発者向け）

- Manifest V3 / content script
- 対象: `*://character-sheets.appspot.com/schedule/*`
- 日程表 DOM 構造は `table.schedule_table` / `#namerow` / `#filterrow` / `#row_*` を参照
- ホワイトリストは `chrome.storage.sync` に `daycodeWhitelist:{sessionKey}` 形式で保存
