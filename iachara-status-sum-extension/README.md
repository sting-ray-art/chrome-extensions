# いあきゃら ステータス合計（Chrome拡張）

[いあきゃら](https://iachara.com) のキャラクターシート画面で、8つの能力値（STR / CON / POW / DEX / APP / SIZ / INT / EDU）を自動で読み取り、合計を画面右下に表示するツールです。

キャラ作成や成長後の確認で「能力値の合計がいくつか」をすぐ知りたいときに使えます。

---

## このツールでできること

- キャラクターシート（閲覧・編集画面）を開くと、右下に「ステータス合計」パネルが表示されます
- 各能力値と、その合計が一覧で確認できます
- シート上の数値を変更すると、自動で合計が更新されます
- 🔄 ボタンで手動更新、— ボタンでパネルの最小化ができます

## 使い方

### 基本操作

1. Chrome で [いあきゃら](https://iachara.com) にログインします
2. キャラクターシートの **閲覧画面** または **編集画面** を開きます
3. 画面右下に **「ステータス合計」** パネルが表示されます
4. パネル内に、各能力値と合計が表示されます

### パネルのボタン

| ボタン | 機能 |
|--------|------|
| 🔄 | 表示を手動で更新します。数値がうまく反映されないときに押してください |
| — | パネルを小さく（最小化）します。再度 ▢ を押すと元の大きさに戻ります |

### 合計の対象

次の 8 つの能力値を合計します。

`STR` / `CON` / `POW` / `DEX` / `APP` / `SIZ` / `INT` / `EDU`

HP・MP・SAN などは含まれません。

## うまく動かないとき

### パネルが表示されない

- いあきゃらの **トップページや一覧ページ** では表示されません。**キャラクターシートの閲覧/編集画面** を開いてください
- ページを再読み込み（F5 キー、または Ctrl+R / Cmd+R）してみてください
- `chrome://extensions` を開き、「いあきゃら ステータス合計」が **有効** になっているか確認してください

### 数値が「-」のまま、または合計がおかしい

- キャラクターシートの読み込みが完了してから、🔄 ボタンを押してみてください
- 別のキャラクターに切り替えた直後は、一瞬古い表示が残ることがあります。🔄 で更新してください

### 「未検出: STR, ...」と表示される

- シート上にその能力値の欄がない、またはまだ値が入っていない可能性があります
- 編集画面で各能力値を入力・保存してから、再度確認してください

## 技術メモ（開発者向け）

- Manifest V3 / content script（`world: MAIN`）
- 対象: `*://iachara.com/*`, `*://www.iachara.com/*`
- DOM のラベル検索と API レスポンスの両方から能力値を取得
- React（Next.js）の SPA 遷移・DOM 再描画に追従するため `MutationObserver` と `history` フックを使用
