# お誕生日.jp 誕生〇 抽出コピー（Chrome拡張）

[お誕生日.jp](https://otanjoubi.jp) の誕生日詳細ページから、誕生花・誕生石・誕生色などの情報をプレーンテキストに整形してクリップボードにコピーするツールです。

キャラクター設定やメモ用に、誕生日にまつわる情報をまとめて取り込みたいときに使えます。

---

## このツールでできること

- 誕生日詳細ページ（`select.php`）を開くと、画面右下に **「誕生〇をコピー」** ボタンが表示されます
- ボタンを1回クリックするだけで、次の項目をまとめてコピーします
- コピー結果はプレーンテキスト形式なので、メモ帳やドキュメントにそのまま貼り付けられます

### 抽出される項目

誕生花 / 誕生石 / 誕生色 / 誕生酒 / 誕生果 / 誕生魚 / 誕生鳥 / 誕生木 / 誕生竜 / 誕生星

## 使い方

### 基本操作

1. Chrome で [お誕生日.jp](https://otanjoubi.jp) の誕生日詳細ページを開きます

```
https://otanjoubi.jp/select.php?month=2&day=6
```

（`month` は月、`day` は日。見たい日付に合わせて変更してください）

2. ページの読み込みが完了するのを待ちます
3. 画面右下の **「誕生〇をコピー」** ボタンをクリックします
4. 「コピーしました。」と表示されたら、メモ帳などに貼り付けて内容を確認します

### コピー結果の形式

各項目が見出し付きのブロックになり、ブロック間は空行で区切られます。例:

```
誕生花
スノードロップ：希望

誕生石
アメジスト：誠実

誕生色
...
```

## うまく動かないとき

### ボタンが表示されない

- URL が `https://otanjoubi.jp/select.php?...` の形式か確認してください（トップページではボタンが出ません）
- ページの読み込みが完了してから確認してください
- `chrome://extensions` を開き、「お誕生日.jp 誕生〇 抽出コピー」が **有効** になっているか確認してください

### 「抽出できませんでした」と表示される

- ページの表示が途中の可能性があります。再読み込み（F5 キー、または Ctrl+R / Cmd+R）してから再度お試しください
- お誕生日.jp のページ構成が変わった場合、抽出に失敗することがあります

### コピーに失敗する

- ブラウザのクリップボード権限がブロックされていないか確認してください
- ページを再読み込みして、もう一度ボタンを押してみてください

## 技術メモ（開発者向け）

- Manifest V3 / content script
- 対象: `https://otanjoubi.jp/select.php*`
- `.ms-section .ms-tableCell` 内の `h2` 見出しをキーに各セクションを抽出
- 抽出ロジックは `extract.js` に集約（`content.js` とポップアップから共用可能）
